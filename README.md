# Nova-UI-Extended
Extended Features of Nova UI by Dead Coffee

Features:
- Extended Text Field:
* Character Validation (None, Alphanumeric, Username, Email Address, Password)
* Protected Field Display
* Input Length Limit
* TMP Tags
* Show Protected Button
